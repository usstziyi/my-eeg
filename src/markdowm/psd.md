(X(f)) 是信号 (x(t)) 的**频域表示**，也就是它的**傅里叶变换**。

如果 (x(t)) 是时域信号，那么：

[
X(f)=\int_{-\infty}^{\infty} x(t)e^{-j2\pi ft},dt
]

这里：

* (x(t))：时间变化的 EEG 电压
* (f)：频率
* (X(f))：信号在频率 (f) 上有多少“成分”

---

## 直观理解

时域里的 (x(t)) 看起来是一条随时间起伏的波。
傅里叶变换后得到 (X(f))，就是在问：

**这条波里面，包含多少 1 Hz、2 Hz、10 Hz、20 Hz …… 的振荡成分？**

所以：

* 时域 (x(t))：看“波形怎么随时间变化”
* 频域 (X(f))：看“各个频率各有多强”

---

## 为什么 (X(f)) 是复数

(X(f)) 一般不是普通实数，而是**复数**，因为它同时包含：

* **幅度信息**：这个频率有多强
* **相位信息**：这个频率的时间位置/初始相位如何

通常可以写成：

[
X(f)=|X(f)|e^{j\phi(f)}
]

其中：

* (|X(f)|)：幅度谱
* (\phi(f))：相位谱

---

## 和 PSD 的关系

你前面问功率谱密度，PSD 和 (X(f)) 的关系可以粗略理解为：

[
PSD(f)\propto |X(f)|^2
]

意思是：

* (X(f)) 先告诉你这个频率成分有多大
* 再取模平方，就得到和功率相关的量

所以：

* (|X(f)|)：更像“振幅大小”
* (|X(f)|^2)：更像“能量/功率大小”

---

## 举个最简单例子

如果信号是：

[
x(t)=A\cos(2\pi f_0 t)
]

那它的频域 (X(f)) 会主要集中在 (+f_0) 和 (-f_0) 两个位置。
意思是这个信号几乎只由频率 (f_0) 的振荡构成。

比如一个纯 10 Hz 的正弦波，(X(f)) 就会在 10 Hz 附近非常突出。

---

## 在 EEG 里怎么理解

EEG 不是单一正弦波，而是很多频率混在一起：

* 低频漂移
* theta
* alpha
* beta
* 肌电高频噪声
* 工频干扰

所以 EEG 的 (X(f)) 不是单个尖峰，而是一整条频谱。
看 (X(f)) 或 PSD，本质上就是在看：

**这段 EEG 里，各频率成分分别有多强。**

---

## 一句话总结

**(X(f)) 就是时域信号 (x(t)) 经过傅里叶变换后得到的频域表达，表示信号在频率 (f) 上的复幅值。**

你可以把它先记成：

* (x(t))：时间里的波形
* (X(f))：频率里的分解结果

# 2. 每个量分别是什么意思

## (1) (x(t))：时域信号

这是你真正采到的 EEG 波形。

```text
电压
 ^
 |      /\      /\/\      /\
 |     /  \    /    \    /  \
 |____/____\__/______\__/____\____> t
```

它表示：

* 横轴：时间
* 纵轴：电压

你看到的是“波随时间怎么变化”。

---

## (2) (X(f))：频域复谱

对 (x(t)) 做傅里叶变换：

[
X(f)=\int_{-\infty}^{\infty} x(t)e^{-j2\pi ft}dt
]

得到 (X(f))。
它是一个**复数函数**，表示每个频率上有多少成分。

```text
频域：
X(f) = 复数 = 幅度 + 相位
```

也可以写成：

[
X(f)=|X(f)|e^{j\phi(f)}
]

所以 (X(f)) 里面有两部分：

* (|X(f)|)：这个频率有多强
* (\phi(f))：这个频率的相位是多少

---

# 3. 从 (X(f)) 可以拆出什么

---

## (3) 幅度谱 (|X(f)|)

这是把复数的“大小”取出来：

[
|X(f)|=\sqrt{\Re(X(f))^2+\Im(X(f))^2}
]

它表示：

**每个频率成分的振幅有多大**

图像大概像这样：

```text
幅度
 ^
 |             /\
 |            /  \
 |     /\    /    \      /\
 |____/__\__/______\____/__\____> f
```

例如：

* 10 Hz 很高 → 说明 10 Hz 振荡很强
* 20 Hz 很低 → 说明 20 Hz 成分较弱

---

## (4) 相位谱 (\phi(f))

[
\phi(f)=\angle X(f)
]

它表示：

**这个频率成分在时间上的“起始位置”或相位偏移**

相位谱通常不像功率谱那样直观，但在重建信号、相干分析、连接性分析里很重要。

---

## (5) 功率谱 (|X(f)|^2)

对幅度再平方：

[
|X(f)|^2
]

它表示频率成分的“能量强度”或“功率强度”。

```text
功率
 ^
 |              /\
 |             /  \
 |      /\    /    \        /\
 |_____/__\__/______\______/__\___> f
```

和幅度谱相比：

* 形状类似
* 但更强调大峰值
* 单位也不一样

---

# 4. PSD 和功率谱有什么区别

这是最容易混的地方。

## 功率谱

更像是：

**某个频率点或频率bin上有多少功率**

## PSD（功率谱密度）

更像是：

**单位频率宽度内有多少功率**

即：

[
PSD(f) \approx \frac{\text{功率}}{\text{Hz}}
]

所以单位常常是：

[
\mu V^2/Hz
]

---

# 5. 为什么要有“密度”这个概念

因为实际算频谱时，频率不是连续点，而是一个个小频段 bin。

比如：

* 一个 bin 覆盖 9.5–10.0 Hz
* 另一个 bin 覆盖 10.0–10.5 Hz

如果不用“每 Hz 的功率”来表示，那么：

* bin 宽一点，数值就可能更大
* bin 窄一点，数值就可能更小

这样不同设置就不好比较。

所以 PSD 通过“除以频率分辨率”来标准化。

# 6. 一个最直观的例子

假设信号是一个纯正弦波：

[
x(t)=A\cos(2\pi f_0 t)
]

---

## 时域

```text
x(t)
 ^
 |    / \    / \    / \
 |   /   \  /   \  /   \
 |__/____\/____\/____\____> t
```

---

## 频域复谱 (X(f))

它主要集中在 (+f_0) 和 (-f_0)：

```text
X(f)
 ^
 |          |           |
 |          |           |
 |__________|___________|______> f
          -f0          +f0
```

因为实信号的频谱通常正负频率对称。

---

## 幅度谱 (|X(f)|)

```text
|X(f)|
 ^
 |          |           |
 |          |           |
 |__________|___________|______> f
          -f0          +f0
```

---

## 功率谱 (|X(f)|^2)

```text
|X(f)|²
 ^
 |          |           |
 |          |           |
 |__________|___________|______> f
          -f0          +f0
```

只是数值变成平方后的量。

---

## PSD

如果这个信号有噪声，或者你用有限长度窗去估计，理想的“尖线”会变成“峰”：

```text
PSD
 ^
 |                    /\
 |                   /  \
 |__________________/____\____________> f
                    f0
```

在 EEG 里这就像 alpha 峰、beta 峰那样。

# 7. EEG 里这些量怎么理解

对 EEG 来说：

## 时域 (x(t))

你看到的是混合波形：

* 慢漂移
* alpha
* beta
* 噪声
* 伪迹

```text
x(t): 一团复杂起伏的波
```

## 频域 (X(f))

把这团复杂波拆成很多频率成分。

## 幅度谱

看“每个频率有多强”。

## 功率谱 / PSD

看“每个频率上的能量有多大”。

例如：

* 10 Hz 附近高峰 → alpha 强
* 50/60 Hz 尖峰 → 工频干扰
* 高频整体抬高 → 可能有肌电

# 9. 一句话区分

你可以这么记：

* **(x(t))**：原始时间波形
* **(X(f))**：频域复表示
* **(|X(f)|)**：幅度谱
* **(|X(f)|^2)**：功率谱
* **PSD**：单位 Hz 的功率分布

---

# 10. 最后给你一个非常好记的类比

把信号看成“声音”：

* (x(t))：声音波形本身
* (X(f))：把声音拆成不同音调
* (|X(f)|)：每个音调有多响
* (|X(f)|^2)：每个音调的能量
* PSD：每个频率范围内平均有多少能量
